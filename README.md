# 引き寄せの法則 - Developer Website

引き寄せの法則アプリの開発者ウェブサイトです。

## ファイル構成

- `index.html` — メインページ
- `privacy-policy.html` — プライバシーポリシー
- `app-ads.txt` — AdMob広告認証ファイル

## GitHub Pagesでの公開方法

1. このリポジトリをGitHubにプッシュ
2. Settings → Pages → Source: `main` ブランチ / `/ (root)` を選択
3. 公開されたURL（例: `https://username.github.io/law-of-attraction-site/`）をGoogle Playデベロッパーページに登録
